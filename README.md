## Project Overview

This repository contains the complete solution for the **Smith & Nephew SIOP (Sales, Inventory, and Operations Planning) Reporting Case Study**.

The project focuses on transforming raw business data into actionable insights through data validation, transformation, modeling, KPI creation, and interactive dashboard development using **Power BI**.

The objective was to support business decision-making by providing meaningful insights into forecast performance, sales trends, and operational planning metrics.
